Sticky Nav

Hero Swiper

The purpose of this snippet is to create a Bootstrap 4 hero swiper.

          

              <style>
              /* Make the image fully responsive */
              
              .carousel-inner img {
                  width: 100%;
                  height: 100%;
              }
          </style>
      
          <div class="container">
      
              <div id="theCarousel" class="carousel slide" data-ride="carousel">
      
                  <!-- Indicators -->
                  <ul class="carousel-indicators">
                      <li data-target="#theCarousel" data-slide-to="0" class="active"></li>
                      <li data-target="#theCarousel" data-slide-to="1"></li>
                      <li data-target="#theCarousel" data-slide-to="2"></li>
                  </ul>
      
                  <!-- The slideshow -->
                  <div class="carousel-inner">
                      <div class="carousel-item active">
                          <img src="http://placecorgi.com/1100/500" alt="Los Angeles" width="1100" height="500">
                      </div>
                      <div class="carousel-item">
                          <img src="http://placecorgi.com/1100/500" alt="Chicago" width="1100" height="500">
                      </div>
                      <div class="carousel-item">
                          <img src="http://placecorgi.com/1100/500" alt="New York" width="1100" height="500">
                      </div>
                  </div>
      
                  <!-- Left and right controls -->
                  <a class="carousel-control-prev" href="#theCarousel" data-slide="prev">
                      <span class="carousel-control-prev-icon"></span>
                  </a>
                  <a class="carousel-control-next" href="#theCarousel" data-slide="next">
                      <span class="carousel-control-next-icon"></span>
                  </a>
              </div>
      
          </div>

Requirements for Snippet:

  • Bootstrap 4
  • Custom CSS to Merge
